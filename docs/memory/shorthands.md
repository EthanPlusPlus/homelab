## Workflow Shorthands

**"Make this canon"** (or "add this to canon", "canonise this") — run the full doc workflow:
write it up in the correct `~/canon/<project>/docs/` location, re-index, commit, push,
merge docs branch into master (for code repo worktrees). See `runbooks/make-canon.md`.

**"Send it Bobby"** — skip further deliberation and execute the approved plan immediately.
Used when a plan has already been reviewed and the person just wants it done without
additional confirmation steps.
