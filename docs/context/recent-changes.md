- **PostgreSQL + pgvector migration (2026-05-29)** ‚Äî Replaces ChromaDB (docs + code_symbols collections) and both SQLite DBs (workflow.db, code_graph.db) with a single PostgreSQL instance. `db/postgres.py` provides `VectorStore` (ChromaDB-compatible interface, pgvector HNSW cosine indexes), `PgConn` (sqlite3-compatible wrapper so router/review/metrics code is unchanged), and `init_pg_schema()` for all tables. `workflow/db.py` delegates to PostgreSQL when `DATABASE_URL` is set; SQLite fallback preserves test compatibility. `docker-compose.yml` adds `pgvector/pgvector:pg16` service with healthcheck; `chroma_store` volume removed.
